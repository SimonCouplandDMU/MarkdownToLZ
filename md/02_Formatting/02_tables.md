### Table with alignment

| Name   | Left Align | Centre Align | Right Align |
|:-------|:-----------|:------------:|------------:|
| Alice  | Text       | Text         | Text        |
| Bob    | Text       | Text         | Text        |

---

### Nested content (supported)

| Feature     | Example                  |
|-------------|--------------------------|
| List        | - Item 1<br>- Item 2     |
| Code        | `int x = 10;`            |
| Bold text   | **Important**            |

> Note: True nested structures are not supported, but line breaks (`<br>`) can help simulate them.
>

You can add additional styling in custom.css if you want a styling to be applied to all tables.
