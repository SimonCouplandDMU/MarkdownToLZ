## Links

You can create links using Markdown syntax.

### External Links

Example of a link to an external website:

[Link to example.com](https://www.example.com)

### Internal Links

Internal links between files use relative paths, they need to point to the converted markdown file, i.e. the html file produced:

[Go to Basic Text](01_basic_text.html)


### Broken link example

The following link points to a file that does not exist:

[Broken link example](non_existent_file.html)

This demonstrates what happens when a link target is missing.
